# blog.dev

个人博客，用于记录学习笔记、项目和生活。

## 使用

用 Live Server 打开 `index.html` 即可。

## 添加内容

编辑以下文件添加数据：

- `js/data/articles.js` — 文章
- `js/data/works.js` — 项目
- `js/data/timeline.js` — 时间线

每个文件内有注释说明格式。

## 功能

- 文章系统（分类、搜索、分页）
- 小说书库（支持 TXT/EPUB/HTML/MD/PDF）
- 阅读器（主题切换、字体调整、进度记忆）
- 项目展示、时间线、留言板
- 深色/浅色主题切换
- 响应式布局
